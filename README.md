# nft-marketplace
 NFT Marketplace

## Getting Started

### Clone the Repository
```bash
git clone https://github.com/dandev947366/nft-marketplace
```

### Install dependencies

```bash
cd nft-marketplace
```
```bash
npm install
```

### Build and run the project
```bash
npm run start
```

### Navigate to **http://localhost:3000**

### Package Versions
Node: v22.9.0
NPM: 10.8.3
